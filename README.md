## 💻 Repositório

Este projeto é uma aplicação web simples de login e cadastro de usuários, desenvolvida utilizando o framework [Angular](https://angular.dev/).

A aplicação permite que usuários se registrem e façam login utilizando um formulário com validações. Os dados são armazenados localmente no navegador utilizando LocalStorage, simulando um sistema de autenticação simples para fins de estudo e prática de desenvolvimento front-end.

Este projeto foi desenvolvido com o objetivo de praticar conceitos de Angular, como:
- Componentes
- Formulários reativos
- Validação de campos
- Roteamento
- Manipulação de dados no LocalStorage

## 📂 Estrutura do Projeto
Principais componentes:

- Auth → Responsável por login e cadastro
- Home → Tela exibida após login do usuário

## ▶️ Como Executar o Projeto?

#### 1️⃣ Clone o repositório
```bash
git clone https://github.com/Franklin-Barbosa/TelaLoginCadastro
```
#### 2️⃣ Acessar a pasta do projeto
```bash
cd TelaLoginCadastro
```

#### 3️⃣ Instalar as dependências
```bash 
npm install
```

#### 4️⃣ Rodar a aplicação
```bash
ng serve
```
#### 5️⃣ Abrir no navegador
```bash
http://localhost:4200 
```
